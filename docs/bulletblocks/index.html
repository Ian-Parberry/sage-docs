A simple 3D toy using Bullet Physics consisting of various things that you can knock over by throwing soccer balls at them. With enough patience and perseverance you can push everything off the edge of the world. Shapes and models are rendered using GeometricPrimitive and Model from DirectXTK12. The models are input using the built-in function Model::CreateFromSDKMESH which requires that the models be in SDKMESH format. The 3D models are from free3d.com and were converted from OBJ format using DirectXMesh.
